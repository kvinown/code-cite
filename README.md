
# Code Cite - Web Application

## English Version

**Code Cite Web** is an interactive web-based application designed to help academics, students, and developers manage code citations outside their IDE. It serves as a central hub to visualize, edit, and consolidate code references within a project, ensuring that every snippet of borrowed logic is properly attributed.

### Key Features

1. **Project Import:** Upload your project in `.zip` format. The web app automatically reconstructs the file tree and reads existing metadata.
2. **Auto-Citation Parsing:** Automatically scans your source code for `# CODE-CITE:` comments and converts them into manageable citation items in the dashboard.
3. **Interactive Workspace:** * **File Explorer:** Navigate through your project files seamlessly.
   * **Interactive Selection:** Select lines of code directly in the web editor to create new citations.
   * **Jump-to-Source:** Click on any citation to automatically scroll and highlight the exact code block in the editor.
4. **Metadata Export:** Pack your project back into a `.zip` file, complete with an updated `.codesite.json` file for future use.

### How to Use

1. **Upload:** Drag and drop your project `.zip` file (exported from the extension or a raw project) onto the upload screen.
2. **Manage:** Review parsed citations in the right-hand panel. You can edit notes, change source types (URL/AI/Other), or add new citations by dragging over the code.
3. **Export:** Click "Export Code Cite" to download the updated project containing all your citation metadata.

### Fitur Utama

1. **Impor Proyek:** Unggah proyek Anda dalam format `.zip`. Aplikasi web akan merekonstruksi struktur folder secara otomatis dan membaca metadata yang ada.
2. **Parsing Sitasi Otomatis:** Memindai kode sumber Anda secara otomatis untuk mencari komentar `# CODE-CITE:` dan mengubahnya menjadi item sitasi yang dapat dikelola di dashboard.
3. **Workspace Interaktif:**
   * **File Explorer:** Menjelajahi file proyek dengan mudah.
   * **Seleksi Interaktif:** Pilih baris kode langsung di editor web untuk membuat sitasi baru.
   * **Jump-to-Source:** Klik pada sitasi mana pun untuk menggulir dan menyorot blok kode yang tepat di dalam editor secara otomatis.
4. **Ekspor Metadata:** Mengemas kembali proyek Anda ke dalam file `.zip`, lengkap dengan file `.codesite.json` yang telah diperbarui untuk penggunaan di masa mendatang.

### Cara Penggunaan

1. **Unggah:** Seret dan lepaskan file proyek `.zip` (hasil ekspor dari ekstensi atau proyek mentah) ke layar unggah.
2. **Kelola:** Tinjau sitasi yang terdeteksi di panel sebelah kanan. Anda dapat mengedit catatan, mengubah tipe sumber (URL/AI/Lainnya), atau menambah sitasi baru dengan menyorot kode.
3. **Ekspor:** Klik "Export Code Cite" untuk mengunduh proyek terbaru yang berisi semua metadata sitasi Anda.
